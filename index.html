I am a sample app.

Here is how I was born;
  1. Jenkins pulled all of my files from Github and saved it in the Jenkins server.
  2. Jenkins used my war to create a Docker image with the help of Ansible.
  3. Jenkins send my image to Docker Registry using Ansible.
  4. Jenkins called Ansible again to create a container of me in the Docker server.
  5. Find me at http://docker_server/sample/.
